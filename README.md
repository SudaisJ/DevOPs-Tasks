# UETM Launchpad Internship Program - DevOps Portfolio

Welcome to the central repository for my **DevOps | Cloud Computing** tasks completed during the **UET Mardan (UETM) Launchpad Internship Program**. 

This repository serves as a comprehensive log of my progression through enterprise-grade DevOps practices, cloud infrastructure provisioning, configuration management, and continuous integration/continuous deployment (CI/CD) pipelines.

## 🛠️ Technology Stack & Languages
* **Infrastructure as Code (IaC):** Terraform (HCL)
* **Configuration Management:** Ansible
* **Containerization & Orchestration:** Docker, Kubernetes
* **Cloud Provider:** Amazon Web Services (AWS)
* **CI/CD & Automation:** GitHub Actions, Shell Scripting
* **Security & Secrets:** HashiCorp Vault
* **Monitoring & Logging:** ELK Stack
* **Languages:** Shell, HCL, Python, HTML, Dockerfile, JavaScript, Jinja

## 📂 Repository Structure & Progression

Below is a high-level overview of the progression through the internship modules:

* **`task1/` - `task3/`**: Foundations of CI/CD, basic shell automation, and initial Docker configurations.
* **`task4/`**: Security Scanning and Vulnerability Management workflows.
* **`task5/`**: Container Orchestration utilizing Kubernetes.
* **`task7/`**: Infrastructure Configuration Management and Automation using Ansible.
* **`task8/`**: Centralized Logging and Observability with the ELK Stack.
* **`task10/` - `task11/`**: Advanced Secrets Management and credential injection utilizing HashiCorp Vault.
* **`task12/`**: End-to-end AWS Infrastructure Provisioning, Security Group configuration, automated container deployment, and Dynamic DNS routing via Terraform.
* **`.github/workflows/`**: Centralized GitHub Actions pipelines for automated testing, integration, and deployment.
